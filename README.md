## Getting Started

You can use the project by just cloning it and then

#### Run Locally

Clone the project

```bash
  git clone https://github.com/chnoumanejaz/wild-oasis-hotel-management.git
```

Go to the project directory

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
